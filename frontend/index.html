1. OCR (Распознавание)

Сканирует папки data/docs/*/image и отправляет новые изображения на распознавание.

2. Очистка (Cleaning)

Обрабатывает XML-файлы из data/docs/*/xml и создает clean-файлы в data/docs/*/clean.

3. Индексация (RAG)

Векторизует примеры (Few-shot) для поиска. Основные документы индексируются через блок Конфигурации.

4. Конфигурация RAG и Моделей

Выбор модели для эмбеддингов и генерации. Нажмите "Переиндексировать всё", чтобы очистить базу и создать новые векторы через выбранную модель.

Проверка точности извлечения JSON данных через LLM. Сравнивается с эталонными значениями в data/benchmark/references.
