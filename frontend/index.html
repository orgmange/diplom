1. OCR (Распознавание)

Сканирует папки data/docs/*/image и отправляет новые изображения на распознавание.

2. Очистка (Cleaning)

Обрабатывает XML-файлы из data/docs/*/xml и создает clean-файлы в data/docs/*/clean.

3. Индексация (RAG)

Векторизует данные и примеры для поиска.

4. Структурирование (LLM)

Преобразование OCR-текста в JSON с помощью выбранной модели и RAG.
